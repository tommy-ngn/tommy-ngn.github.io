# Contributing

Thanks for your interest in improving these frameworks.

## What's welcome

- **Typo and grammar fixes** — open a PR directly
- **Broken link reports** — open an Issue
- **Questions about the frameworks** — open a Discussion
- **Suggestions for new content** — open a Discussion
- **Translations** — open a PR with the translated file in a `lang/` subfolder

## What's not accepted

- **Content rewrites or additions to core frameworks** — these frameworks reflect a specific methodology and voice. Suggest changes via Discussion instead.
- **New frameworks or skills** — propose via Discussion first.

## How to submit a fix

1. Fork the repo
2. Make your change
3. Open a PR with a one-line description of what you fixed

That's it. No issue template required for typo fixes.
