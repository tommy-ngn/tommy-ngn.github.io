# Tommy Ng — PM & AI Frameworks

Product frameworks for PMs, founders, and AI builders. Each framework is a system — not a blog post.

**Live site:** [tommy-ngn.github.io](https://tommy-ngn.github.io)

## Frameworks

### [Product Sense & Design](https://tommy-ngn.github.io/product-sense/)
7-phase, 6-level system for deciding what to build before you write a line of code. From clarifying the real problem to designing solutions that compound.

### [AI Decision & Research Skills](https://tommy-ngn.github.io/pm-skills/)
Drop-in AI skills (1-3-1 Analysis, 1-5-X Analysis, Deep Research) that turn any LLM into a structured thinking partner. Criteria before options. Evidence before opinions.

### [AI-Native PM](https://tommy-ngn.github.io/ai-native-pm/) *(coming soon)*
The full architecture behind an AI-native PM workflow — project memory, knowledge indexing, skill orchestration.

### [Claude-ometer](https://tommy-ngn.github.io/claude-ometer/) *(coming soon)*
Scoring model for measuring AI-native PM maturity. Diagnostic + improvement path.

### [Business Acumen](https://tommy-ngn.github.io/business-acumen/) *(coming soon)*
Structured thinking for how businesses create, capture, and compound value.

## Using the AI Skills

Each skill in `pm-skills/` is a self-contained `SKILL.md` you can drop into any AI platform:

**Claude Projects:** Upload `SKILL.md` to Project Knowledge.

**Claude Code / Cowork:**
```bash
cp -r pm-skills/1-3-1-analysis ~/.claude/skills/
cp -r pm-skills/1-5-x-analysis ~/.claude/skills/
cp -r pm-skills/deep-research ~/.claude/skills/
```

**Other platforms:** Add `SKILL.md` content to your system prompt or project context.
